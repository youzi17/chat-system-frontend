# 角色头像目录

这个目录用于存放角色头像图片文件。

## 默认头像

- `default-avatar.png` - 默认头像
- `user-avatar.png` - 用户头像
- `anime-avatar.png` - 动漫角色头像
- `game-avatar.png` - 游戏角色头像

## 使用说明

1. 将头像图片放在此目录下
2. 支持的格式：PNG, JPG, JPEG, SVG
3. 建议尺寸：64x64px 或 128x128px
4. 文件名使用小写字母和连字符

## 示例

```typescript
// 在组件中使用
const avatarUrl = '/role-avatars/anime-avatar.png'
```
